One-step Metropolis-Hastings update.

Parameters
rngRNG engine
xThe current state value. It will be updated to the new value after the Metropolis-Hastings move if it is accepted, and left unchanged otherwise.
ltxIf it is a non-null pointer, then it points to the value of the \(\log\gamma(x)\). It will be updated to the new value if the Metropolis-Hastings move is accepted, and left unchanged otherwise. If it is a null pointer, then it is ignored. Use this pointer to save \(\log\gamma(x)\) between updates if it is expensive to calculate.
log_targetThe log-target fucntion
ReturnType log_target(const result_type *x);
where ReturnType needs to be convertible to double. It accepts a pointer to the states and returns the value of log-target function \(\log\gamma(x)\).
proposalThe proposal function
ReturnType proposal(
RNGType &rng, const result_type *x, result_type *y);
where ReturnType needs to be convertible to double. It accepts a pointer to the states and returns the value of the proposal new states in y and the value of \(\log(q(y, x) / q(x, y))\).
Returns
Acceptance count
