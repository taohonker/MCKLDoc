Detailed Description

template<typename T>
class mckl::ParticleIndex< T >

A thin wrapper over a complete Particle.

This is the basic ParticleIndex available for any type of Particle. To extend it for type T. One can either specialize mckl::ParticleIndexBaseTypeTrait<T> or define a class template named particle_index_type within T with the following minimum requirement.

template <typename S> // S: StateType, such as StateMatrix<Dim, T>
class particle_index_type
{
public:
particle_index_type(
typename Particle<T>::size_type id, Particle<S> *pptr);
size_type id() const;
Particle<S> &particle() const;
Particle<S> *patricle_ptr() const;
};

Usually you can safely derive particle_index_type<S> from ParticleIndexBase<S> and add methods specific to S.
