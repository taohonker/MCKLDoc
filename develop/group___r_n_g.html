RealType mckl::u01 ( UIntType  u)
inlinenoexcept

Convert uniform unsigned integers to floating points within [0, 1].

Let \(W\) be the number of digits of unsigned integer type UIntType. Let \(M\) be the number of significant digits of floating point type RealType. Assuming the input is a uniform random number on the set \(\{0,1,\dots,2^W - 1\), the output is uniform over the interval \([0,1]\) or one of its (half-)open interval variant. The exact output depend on the template parameter Lower and Upper.
